## Struktur Projek 
* **_bin_**  berisi file "*.class"
* **_doc_**  berisi laporan
* **_lib_** berisi jar file
* **_src_**  berisi file "*.java" 
* **_test_** berisi file tes "*.txt"

## Cara Run Program
* Windows
<br> 1. Buka folder lib
<br> 1. Run run.bat
* Terminal
<br> 1. Open CMD ./src
<br> 2. Ketikkan javac -d ../bin ./mtrx/*.java
<br> 3. Ketikkan java -cp ../bin Main
